## プロジェクト紹介
* 留学生のためのアルバイト検索プラットフォーム。
* システムが含まれています：アプリケーション（アンドロイド/ iOS）、ウェブ募集管理、APIサービス。
* スマートフォンアプリでユーザーはアルバイトジョブを検索できること
* Webシステムでリクルーターは、応募者情報を管理できること
